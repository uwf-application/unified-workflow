# pkg/client/typescript/src/

TypeScript SDK source files.

## Files

| File | What | When to read |
| ---- | ---- | ------------ |
| `client.ts` | TypeScript workflow client: execute, register, poll status | Modifying TypeScript client methods |
| `config.ts` | SDK configuration types and defaults | Modifying TypeScript SDK configuration |
| `models.ts` | TypeScript types for workflow requests and responses | Adding or modifying SDK data types |
| `errors.ts` | SDK error classes and error handling | Adding error types, modifying error handling |
| `index.ts` | SDK public exports | Modifying the public SDK surface |
