# api/

Smithy API definition files specifying REST API contracts.

## Files

| File | What | When to read |
| ---- | ---- | ------------ |
| `workflow-api.smithy` | Smithy model for the workflow REST API (operations, shapes, errors) | Modifying API contracts, adding endpoints, understanding request/response shapes |
| `workflow-sdk.smithy` | Smithy model for the workflow SDK client interface | Modifying SDK contracts, understanding client API shape |
